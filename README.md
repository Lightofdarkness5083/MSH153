LLM모델을 비교.선정하기 (문서 1)
 Q&A IBS방식에 의한 성경묵상은 LLM모델을 이용하는 것이 효과적이다. 3가지 LLM모델에서 이 성경묵상을 구현하여 얻어진 묵상 요약을 비교 분석한 자료 와 실제 각 LLM모델을 통하여 Q&A IBS방식에 의한 성경묵상에 최적화 된 모델을 선정하였다.
 
시스템 설계문서 (문서 2)
  선정한 LLM 모델에서 v1(클로드3)에서 V2(클로드5)로 진행한 프롬프트를 주고  
  타겟 사용자, 페르소나, 관찰 해석시 질문 방식을 요철했을 때 어떤 변화가 있는지 확인하도록 요청하여 생성한 문서

실행 로그 (대화 로그) (문서 3)
  선정한 LLM모델에서 Q&A IBS 성경 묵상을 진행할 때 10턴의 질문을 통하여 문제 발생 지점을 발견 수정하는 문서
