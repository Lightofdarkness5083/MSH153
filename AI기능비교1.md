# 골로새서 1:9-14 IBS 묵상 전체 요약 📖

---

## 1단계: 문맥 확인 (1:3-8)

```
바울이 골로새 교회에 대해 들은 좋은 소식:
  ✅ 그리스도 안에서의 믿음
  ✅ 성도들에 대한 사랑
  ✅ 하늘에 쌓인 소망
  → 이 소식이 9절 기도의 동기가 됨
```

---

## 2단계: 본문 관찰 (절별 분석)

### 9절 - 기도의 시작
| 항목 | 내용 |
|------|------|
| **"이로써"** | 1:3-8의 좋은 소식이 기도의 동기 |
| **기도 주체** | 바울과 디모데 |
| **기도 내용** | 하나님의 뜻을 아는 것으로 채워지길 |
| **수단** | 모든 신령한 지혜와 총명 |

### 10절 - 기도의 세 가지 결과
```
주께 합당하게 행하여
  │
  ├─ ① 범사에 기쁘시게 하고
  ├─ ② 모든 선한 일에 열매를 맺게 하고
  └─ ③ 하나님을 아는 것에 자라게 하고
  
※ 순서: 행동(원인) → 결과가 순차적으로 이어짐
```

### 11절 - 기도의 능력
```
하나님의 영광의 힘
  │
  └─ → 모든 견딤과 오래 참음에 이르게 함
  
핵심: 견딤의 근원 = 내 의지가 아닌 하나님의 힘
```

### 12절 - 감사
```
감사 대상: 아버지 하나님
감사 이유: 빛 가운데 성도의 기업의 부분을
          얻기에 합당하게 하셨음
의미: 자격이 아닌 은혜로 주어진 것 → 감사의 근거
```

### 13절 - 하나님의 두 가지 행동
```
행동 ①: 흑암의 권세에서 건져내심 (구출)
행동 ②: 사랑의 아들의 나라로 옮기심 (이동)

흐름: 어둠 → 빛, 옛 영역 → 새 영역
```

### 14절 - 아들 안에서의 결과
```
아들 안에서 얻은 것:
  속량 = 죄 사함

※ 14절은 13절의 원인
  → 죄 사함(속량)이 있었기에
  → 흑암에서 건짐 + 아들 나라로 옮김이 가능했음
```

---

## 3단계: 전체 흐름 정리

```
좋은 소식 (1:3-8)
  │ "이로써"
  ▼
기도 (9절): 하나님의 뜻을 알기를
  │
  ▼
기도의 결과 (10절): 합당한 행동 → 기쁨, 열매, 성장
  │
  ▼
기도의 능력 (11절): 영광의 힘 → 견딤과 오래 참음
  │
  ▼
감사의 이유 (12절): 기업에 합당하게 하심
  │
  ▼
하나님의 행동 (13절): 흑암에서 건짐 → 아들 나라로 옮김
  │
  ▼
그 근거 (14절): 아들 안에서의 속량 = 죄 사함
```

---

## 4단계: 적용 & 기도

> **적용:** 누구를 대하든지 그들에게 있는 **믿음·소망·사랑을 먼저 보고** 칭찬과 감사를 드리겠다

> **기도:** "주님, 골로새서를 읽고 묵상할수록 저자를 통한 하나님의 마음을 알고, 저도 타인을 대할 때 믿음, 소망, 사랑을 보게 하소서" 🙏
